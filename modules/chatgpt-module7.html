ChatGPT Mastery - Module 7

Advanced Prompt Engineering

In this module you will learn advanced prompting techniques to get professional results from ChatGPT.

Lesson 1: Advanced Prompts

Advanced prompts give ChatGPT a clear role, goal, context and output format.

Lesson 2: Prompt Framework

  • Role - Tell AI who it should act as
  • Task - Explain what you need
  • Context - Provide background information
  • Format - Define the output style
  • Example - Give examples when needed

Lesson 3: Professional Prompt Examples

Example Prompt: "Act as an SEO expert and create a complete keyword research plan for my website."

Lesson 4: Improving AI Results

  • Ask follow-up questions
  • Request improvements
  • Add more details
  • Check and edit the final result
